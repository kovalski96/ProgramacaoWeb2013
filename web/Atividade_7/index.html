Introdução ao CSS

Css é o acronimo de Cascading Style Sheets(Folhas de Estilos em Cascatas) e foi desenvouvido para definir como as tags HTML serão mostradas. A necessidade da criação do CSS vem de um problema gerada nas novas tagas do HTML 3.2, onde surgiu tags de formatação como a tag FONT. Entretanto, HTML nunca foi projetado para formatação definida, mesmo que fossem paginas iguais.

Com esse problemas, foi criado o CSS, no qual podemos definir em um aquivo css, as formatações necessarias e em cada pagina web, usara ess aquivo para definnir suas formatações. Então, caso exista necessidade de alterar a aparencia ou o layout da pagina, pasta alterar o arquivo css e todas a paginas serão alteradas.

Para curiosidade, as tags de formatação, como a font continuaram existindo no HTML4, entretanto foram excluidas no HTML%, sendo CSS a unica forma de formatação na nova verção do HTML.
