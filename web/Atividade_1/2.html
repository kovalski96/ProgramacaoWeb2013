Tags (etiquetas)

Todo Documento HTML apresenta elementos entre parênteses angulares (< e >). Esses elementos são as Tags de HTML, que são os comandos de formatação da linguagem. A maioria das Tags tem sua correspondente de fechamento, representada com uma “barra” ( / ):

Algumas tags e suas definições

Tags de estruturação básica de uma pagina

  • HTML - Esta TAG delimita um texto html, ou seja, todo o codigo HTML deve estar dentro das delimitações da tag HTML
  • HEAD - Esta TAG delimita o cabeçario da página, o cabeçario possui a função de ficar com as configurações específicas para o navegador.
  • BODY - Esta TAG delimita o conteúdo da página, ou seja tudo que for conteudo é incerido entre as Tags BODY

Tags para edisão basica de textos

  • H1 até H6 - Estas TAGs Servem para colocar titulos e sub-tilulos com até seis niveis de difernça comessadno pela tag H1 e posteriormente até H6.
  • P - Esta TAG Adiciona um novo paragrafo.
  • Br - Esta TAG Força a quebra de linha em um texto (esta é um exemplo de tag que não precisa de fechamento).
  • EM - Esta TAG Adiciona emfase ao texto.
  • STRONG - Esta TAG Adiciona importancia ao texto.
